1) Посмотреть где я - pwd
2) Создать папку - mkdir [foldername]
3) Зайти в папку - cd [foldername]
4) Создать 3 папки - mdir [foldername1] [foldername2] [foldername3]
5) Зайти в любоую папку - cd [foldername]
6) Создать 5 файлов (3 txt, 2 json) - touch [filename.txt]
7) Создать 3 папки
8) Вывести список содержимого папки - ls
9) + Открыть любой txt файл - vim [filename]
10) + написать туда что-нибудь, любой текст. - insert(i)
11) + сохранить и выйти - esc, :wq
12) Выйти из папки на уровень выше cd ..
13) переместить любые 2 файла, которые вы создали, в любую другую папку. 
   - mv folder_1/[filename folder_2 
либо _ mv folder_1/*.json folder_2 (все файлы с указанным расширением)
14) скопировать любые 2 файла, которые вы создали, в любую другую папку.
  - cp folder_1/test_1.txt folder_2
15) Найти файл по имени - find . -name "[filename.txt]"
  
16) просмотреть содержимое в реальном времени
  -  cat [filename.txt] (содержимое полностью)
  -  grep "искомое слово" [filename.txt] (поиск слова в файле)
17) вывести несколько первых строк из текстового файла
	- tail -n+[номер первой строки] [filename.txt]

18) вывести несколько последних строк из текстового файла
	- tail -n[количество последних строк] [filename.txt]

19) просмотреть содержимое длинного файла (команда less) изучите как она работает.
	- less [filename.txt] (полное содержимое)
	- less -s [filename.txt] (содержимое без пустых строк)
20) вывести дату и время
	- date
